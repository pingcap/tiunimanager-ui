# TiEM UI

## Scripts

```shell
# Clone
git clone https://github.com/pingcap-inc/tiem-ui
cd tiem-ui

# Install Dependencies
yarn

# Bootstrap
yarn bootstrap

# Generate Glue Codes
yarn generate

# Start Dev Server
yarn dev

# Start Dev Server with Mock
yarn dev:mock

# Start Dev Server with WDYR
yarn dev:wdyr

# Lint
yarn lint --fix

# Build
yarn build

# Preview in Production Mode
yarn build && yarn serve
```
